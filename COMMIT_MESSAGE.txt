1. BLE Layer (form/src/main/java/org/dhis2/sensor/ble/)
   - BleScanner.kt: Added Blood Pressure service UUID detection (0x1810)
   - BleDeviceConnector.kt: Implemented BP service subscription with INDICATE mode
   - BleDataParser.kt: Complete IEEE-11073 SFLOAT parser with BP packet parsing
   - KnownDevices.kt: Registered FORA D40b MAC address (C0:26:DA:19:D4:FE)
   - SensorType.kt: Added BLOOD_PRESSURE enum value

2. Configuration Layer (form/src/main/java/org/dhis2/sensor/config/)
   - SensorConfigModels.kt: New multi-measurement architecture with measurements map
   - SensorConfigRepository.kt: Enhanced to support multi-measurement lookups

3. Data Flow
   - User clicks "Connect Blood Pressure" button
   - App scans for BP service UUID or known MAC address
   - Auto-connects when FORA D40b is discovered
   - Subscribes to BP Measurement characteristic (0x2A35)
   - User performs measurement on physical device
   - App receives BLE notification packet
   - Parser extracts systolic, diastolic, and optional pulse rate
   - ViewModel auto-populates three DHIS2 data element fields

TECHNICAL SPECIFICATIONS
========================

Blood Pressure Packet Format (Bluetooth SIG):
- Byte 0: Flags (units, optional fields)
- Bytes 1-2: Systolic (IEEE-11073 SFLOAT)
- Bytes 3-4: Diastolic (IEEE-11073 SFLOAT)
- Bytes 5-6: MAP (IEEE-11073 SFLOAT)
- Bytes 7+: Optional (timestamp, pulse rate, user ID, status)

IEEE-11073 16-bit SFLOAT:
- Bits 0-11: 12-bit signed mantissa (two's complement)
- Bits 12-15: 4-bit signed exponent (two's complement)
- Value = mantissa × 10^exponent
- Special values: NaN (0x07FF), NRes (0x0800), ±INFINITY

Supported Features:
- mmHg and kPa units (automatic conversion)
- Systolic pressure (50-250 mmHg range validation)
- Diastolic pressure (30-150 mmHg range validation)
- Pulse rate (30-250 bpm range validation)
- Timestamp parsing (skipped)
- User ID parsing (skipped)
- Measurement status parsing (skipped)

ARCHITECTURE CHANGES
====================

New Multi-Measurement Structure:
```json
{
  "name": "Blood Pressure",
  "type": "multi",
  "serviceUUID": "00001810-0000-1000-8000-00805f9b34fb",
  "characteristicUUID": "00002A35-0000-1000-8000-00805f9b34fb",
  "macAddress": "C0:26:DA:19:D4:FE",
  "measurements": {
    "systolic": {"dataElement": "HkfzcXMdLLF", "unit": "mmHg"},
    "diastolic": {"dataElement": "skBarAsIYIL", "unit": "mmHg"},
    "pulse": {"dataElement": "tZbUrUbhUNy", "unit": "bpm"}
  }
}
```

Legacy Support (Backward Compatible):
- Single-value: dataElement + unit
- Dual-value: dataElements (systolic/diastolic only)
